# React Router

## Routing이란?

화면에 따라 '주소'도 달라지는데 이렇게 다른 주소에 따라 다른 뷰를 보여주는 과정을 '경로에 따라 변경한다'라는 의미로 라우팅이라고 합니다.
하지만 React 자체에는 이 기능이 내장되어 있지 않기 때문에 라우팅을 위해 React ROuter이라는 라이브러리를 가장 많이 사용합니다.
